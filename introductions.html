Introductions


- TCP/IP:

  Is a cluster of protocols that include the Transmission Control Protocol/Internet Protocol, File Transfer Protocol, Simple Mail Transfer Protocol and the User Datagram Protocol.

- P2P: Peer to Peer

  Is the payment between individuals.

- WWW: World Wide Web

  Is a system of hypertexts to let browser access a web server in internet through URI.

- URI: Uniform Resource Identifier

  Is a set of string to identify and let users to locate a local or internet resource through a browser.

- Web 1, 2, 3.0

  -Web 1.0: Websites provide read-only resources, users wouldn't change the website.

  -Web 2.0: Users would interact between each others and upload resources to the websites (bbs, blog).

  -Web 3.0: Actually there isn't a certain definition of Web 3.0, but artificial intelligence and blockchain are commonly considered as elements in the web 3.0 so far.

- FTP: File Transfer Protocol

  Is a set of standarized protocols to ensure the reliable data transmission between the host and user.
